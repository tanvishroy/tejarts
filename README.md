🎨 Tej Arts & Crafts Institute Website

A modern, responsive website for Tej Arts & Crafts Institute, showcasing creative art, craft, and handwriting courses.
The site highlights institute offerings with engaging visuals, dropdown navigation, and a Swiper-powered hero carousel.

🖼️ Overview

This project is a front-end website designed to display the various courses offered by Tej Arts & Crafts Institute.
It includes:

A fixed navigation bar with dropdown menus.

A full-width responsive image carousel with autoplay and progress indicators.

A grid layout of art and craft courses.

A contact section with location, phone numbers, and YouTube channel links.

Mobile-friendly design with a collapsible menu and interactive elements.

🧱 Technologies Used
Technology	Purpose
HTML5	Structure of the website
CSS3 (with Google Fonts & Media Queries)	Styling and responsive layout
JavaScript (Vanilla)	Dropdown menus, slide transitions, and animations
Swiper.js	Hero carousel functionality
Font Awesome	Icons for contact info and YouTube links
⚙️ Features

🧭 Responsive Navbar with dropdown menus and mobile toggle.

🖼️ Swiper Carousel with autoplay and custom progress bar.

🎓 Course Showcase Grid with hover effects and frosted glass overlays.

📱 Mobile-Friendly Layout across all screen sizes.

📍 Contact Section with Google Maps, phone, and email integration.

▶️ Floating YouTube Button with animated pulse effect.
